## SMB - Server Message Block

Windows file sharing protocol. **SMB** runs on **TCP port 445** (modern) or port 139 (legacy NetBIOS). **CIFS** (Common Internet File System) is an older version of the same protocol.

---

## Protocol Quick Reference

| Protocol | Full Name | Purpose | Default Port |
|----------|-----------|---------|--------------|
| **SMB** | Server Message Block | File sharing | `445` |
| **FTP** | File Transfer Protocol | File transfer | `21` |
| **SSH** | Secure Shell | Remote login (encrypted) | `22` |
| **HTTP** | HyperText Transfer Protocol | Websites | `80` |
| **SQL** | Structured Query Language | Database queries | `3306` (MySQL) |

---

## Nmap — Filter Open Ports Only

```bash
nmap <target> --open
# Only shows ports in "open" state
# Filters out closed/filtered ports for cleaner output
```

---

## Windows — Map/Unmount SMB Shares

```bash
# Delete existing mapped drive
net use z: /delete

# Map SMB share with credentials
net use z: \\192.168.177.129\opt /user:msfadmin msfadmin

# Map SMB share with blank/null credentials
net use z: \\192.168.177.129\opt "" /user:""
```

| Command | Purpose |
|---------|---------|
| `net use z: /delete` | Unmount drive `Z:` |
| `net use z: \\IP\share /user:username password` | Mount with creds |
| `net use z: \\IP\share "" /user:""` | Mount with null session |

---

## Key Takeaways

| Item | Detail |
|------|--------|
| **SMB port** | `445` (modern), `139` (legacy) |
| **CIFS** | Older name for SMB — same protocol family |
| **`--open`** | Nmap flag to show only open ports |
| **`net use`** | Windows CLI for mounting network shares |
| **Null session** | `"" /user:""` — anonymous SMB access |
