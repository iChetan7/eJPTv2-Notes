## Tcpdump / Wireshark

Packet capture and network analysis tools for traffic inspection and troubleshooting.

---

## ARP — Address Resolution Protocol

Maps **IP addresses to MAC addresses** on a local network.

```bash
# ARP scan — discover hosts on local network
sudo arp-scan -I eth0 192.168.177.128/24

# With range
sudo arp-scan -I eth0 -g 192.168.1.1/24
```

---

## ICMP — Internet Control Message Protocol

Used for **diagnosing network connectivity issues**.

**Tool:** Traceroute

---

## PING

Sends **Type 8 Echo Request** to test host reachability.

```bash
# Basic ping
ping 192.168.1.1

# Check interface
ip a
```

---

## Wireshark

GUI packet analyzer. View traffic by protocol, source, destination.

**Navigation:** Statistics → Endpoints (to see all communicating hosts)

---

## FPING

Fast ping scanner for multiple hosts.

```bash
# Scan entire subnet
fping -I eth0 -g 192.168.1.0/24 -a

# Suppress errors
fping -I eth0 -g 192.168.1.0/24 -a 2>/dev/null
```

---

## Standard Error Redirection

| Redirect | Meaning |
|----------|---------|
| `2> /dev/null` | Keep good results, trash errors |
| `1> /dev/null` | Keep errors, trash good results |
| `&> /dev/null` | Trash everything (total silence) |

## Nmap

Open-source tool for network exploration and security auditing. Uses raw IP packets to determine:

- Live hosts on the network
- Services (name and version) running
- Operating systems and versions
- Packet filters/firewalls in use

---

## Zenmap

Multi-platform **graphical Nmap frontend**.

| Feature             | Description                      |
| ------------------- | -------------------------------- |
| Profiles            | Save frequently used scans       |
| Command Creator     | Interactive Nmap command builder |
| Save Results        | Store and view scans later       |
| Compare Scans       | Diff between scan results        |
| Searchable Database | Recent scan history              |
