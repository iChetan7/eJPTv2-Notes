## First Exercise - Basic Network Recon

Step-by-step host discovery and port scanning workflow.

---

## 1. Ping

Test basic host reachability.

```bash
ping 192.168.1.1
```

---

## 2. ARP-Scan

Discover live hosts on local network segment.

```bash
sudo arp-scan -g 192.168.1.1
```

---

## 3. Nmap

Default port scan on target.

```bash
nmap 192.168.1.1
```

Skip host discovery if target appears down (treat as online).

```bash
nmap -Pn 192.168.1.1
```

Scan specific ports only.

```bash
nmap -Pn -p 21,22,23,80,8082 192.168.1.1
```

---

## Command Reference

| Command       | Purpose                              |
| ------------- | ------------------------------------ |
| `ping`        | ICMP echo request — host alive check |
| `arp-scan -g` | ARP discovery on local subnet        |
| `nmap`        | Default top 1000 ports scan          |
| `nmap -Pn`    | Skip ping probe, scan regardless     |
| `nmap -p`     | Specify custom port list             |
