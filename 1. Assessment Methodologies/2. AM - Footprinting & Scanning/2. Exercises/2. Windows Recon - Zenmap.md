## NSE - Nmap Scripting Engine

Built-in scripting engine for advanced scanning, vulnerability detection, and service enumeration.

---

## DHCP - Dynamic Host Configuration Protocol

Assigns IP addresses dynamically to devices on a network.

---

## Scan Types

### 1. Normal Scan (Top 1000 ports)

```bash
nmap 192.148.40.3
```

Scans the **top 1000 most common ports** — fast and efficient for quick recon.

---

### 2. Full Scan (All 65535 ports)

```bash
nmap 192.148.40.3 -p-
```

Scans **all 65,535 TCP ports** — thorough but slower. Use when normal scan misses services.

---

## Flag Reference

| Flag      | Meaning                  |
| --------- | ------------------------ |
| `-p-`     | Scan all ports (1-65535) |
| (no flag) | Default top 1000 ports   |
