## SMB Enumeration with Nmap NSE

SMB (Server Message Block) runs on **TCP port 445**. Nmap Scripting Engine (NSE) provides dedicated scripts for SMB reconnaissance, security assessment, and enumeration.

---

## smb-protocols

Identifies which SMB protocol versions the target supports.

**Categories:** safe, discovery

```bash
nmap -p445 --script smb-protocols 192.168.177.129
```

---

## smb-security-mode

Determines the security configuration of the SMB server. Tells you how secure the running version is configured.

```bash
nmap -p445 --script smb-security-mode 192.168.177.129
```

---

## smb-enum-sessions

Enumerates users currently logged in — locally or via SMB shares.

```bash
nmap -p445 --script smb-enum-sessions --script-args smbusername=msfadmin,smbpassword=msfadmin 192.168.177.129
```

---

## smb-enum-shares

Lists all shared folders and checks permission levels for each.

```bash
# Anonymous
nmap -p445 --script smb-enum-shares 192.168.177.129

# Authenticated
nmap -p445 --script smb-enum-shares --script-args smbusername=msfadmin,smbpassword=msfadmin 192.168.177.129

# With file listing
nmap -p445 --script smb-enum-shares,smb-ls --script-args smbusername=msfadmin,smbpassword=msfadmin 192.168.177.129
```

> **IPC$** (Inter-Process Communication) is a hidden virtual share — critical target for attackers.

---

## smb-enum-users

Scrapes all local user accounts from the target.

```bash
nmap -p445 --script smb-enum-users --script-args smbusername=msfadmin,smbpassword=msfadmin 192.168.177.129
```

---

## smb-server-stats

Pulls performance and activity data from the SMB server.

```bash
nmap -p445 --script smb-server-stats --script-args smbusername=msfadmin,smbpassword=msfadmin 192.168.177.129
```

---

## Additional SMB Scripts

| Script | Purpose |
|--------|---------|
| smb-enum-domains | Enumerates Domain info, Windows Workgroups, Browser Service |
| smb-enum-groups | Lists security groups and their members |
| smb-enum-services | Lists all active services on the target |

---

## Script Reference

| Script            | Requires Auth | Info Gathered               |
| ----------------- | ------------- | --------------------------- |
| smb-protocols     | No            | SMB version support         |
| smb-security-mode | No            | Security settings           |
| smb-enum-sessions | Yes           | Logged-in users             |
| smb-enum-shares   | Optional      | Shared folders, permissions |
| smb-enum-users    | Yes           | Local user accounts         |
| smb-server-stats  | Yes           | Server performance data     |
| smb-enum-domains  | Yes           | Domain/Workgroup info       |
| smb-enum-groups   | Yes           | Security groups             |
| smb-enum-services | Yes           | Running services            |
