## SMBMap & SMBClient

Tools for enumerating and interacting with SMB shares — alternative to Nmap NSE scripts for deeper share access and file operations.

---

## SMBMap

Enumerates Samba share drives across an entire domain. Use `-d .` for **local machine** targeting with a local admin or standard user account.

### Anonymous / Guest Access
```bash
smbmap -u guest -p "" -d . -H 192.168.177.129
```

### Authenticated Access
```bash
smbmap -u msfadmin -p msfadmin -d . -H 192.168.177.129
```

### Execute Remote Command
```bash
smbmap -u msfadmin -p msfadmin -d . -H 192.168.177.129 -x 'ipconfig'
```

### List Contents
```bash
smbmap -u msfadmin -p msfadmin -d . -H 192.168.177.129 -x -L
```

### Recursive Share Listing
```bash
smbmap -u guest -p '' -d . -H 192.168.177.129 -r 'C$'
```

### Upload File
```bash
smbmap -u guest -p '' -d . -H 192.168.177.129 --upload '/root/test1' 'C$/test1'
```

### Download File
```bash
smbmap -u guest -p '' -d . -H 192.168.177.129 --download 'C$/test1'
```

## SMBClient

Part of the standard Samba suite. Best fallback when SMBMap crashes (Python compatibility issues). Handles null/no-password sessions perfectly.

### List Shares
```bash
smbclient -L 192.168.177.129 -U msfadmin%msfadmin
```

| Flag | Meaning |
|------|---------|
| `-L` | List shares |
| `-U` | Specify username and password (`user%pass`) |

---

## Flag Reference

| Flag | Tool | Purpose |
|------|------|---------|
| `-u` | SMBMap | Username |
| `-p` | SMBMap | Password |
| `-d .` | SMBMap | Local domain (target specific machine) |
| `-H` | SMBMap | Target host IP |
| `-x` | SMBMap | Execute command |
| `-r` | SMBMap | Recursive listing |
| `--upload` | SMBMap | Upload file to share |
| `--download` | SMBMap | Download file from share |
| `-L` | SMBClient | List available shares |
| `-U` | SMBClient | Username%password |

---

## When to Use What

| Scenario                       | Tool      | Reason                       |
| ------------------------------ | --------- | ---------------------------- |
| Quick share enumeration        | SMBMap    | Clean output, domain-wide    |
| SMBMap crashes / Python issues | SMBClient | Stable, standard Samba suite |
| Null session / no password     | SMBClient | Handles anonymous better     |
| File upload/download           | SMBMap    | Built-in file operations     |
| Remote command execution       | SMBMap    | `-x` flag                    |
