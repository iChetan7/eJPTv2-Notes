SMB is the **protocol** (language). **Samba** is the **software** (translator) that lets Linux and Windows computers communicate.

---

## SMB Enumeration Workflow

### 1. Nmap Scan

```bash
# Check SMB protocol versions
nmap 192.168.177.129 -p445 --script smb-protocols

# Enumerate user accounts
nmap 192.168.177.129 -p445 --script smb-enum-users
```

### 2. Service Version Scan

```bash
nmap 192.168.177.129 -sV -p 139,445
```

### 3. UDP Scan

```bash
# Top 20 UDP ports
nmap 192.168.177.129 -sU --top-ports 20

# With version detection and open filter
nmap 192.168.177.129 -sU --top-ports 20 --open -sV
```

### 4. OS Discovery

`smb-os-discovery` retrieves OS version, computer name, domain, workgroup, and system time — often without credentials.

```bash
nmap 192.168.177.129 -p 445 --script smb-os-discovery
```

## Smbclient

List shares with anonymous access.

```bash
smbclient -L 192.168.177.129 -N
```

| Flag | Meaning |
|------|---------|
| `-L` | List shares |
| `-N` | No password (anonymous) |

## Rpcclient

Remote Procedure Call client for SMB — powerful for information gathering without full SMB session.

```bash
rpcclient -U "" -N 192.168.177.129
```

| Flag | Meaning |
|------|---------|
| `-U ""` | Empty username |
| `-N` | No password |

### Rpcclient Commands

```bash
rpcclient $> ?
# List all available commands

rpcclient $> srvinfo
# Server info

rpcclient $> enumdomusers
# List domain users

rpcclient $> lookupnames admin
# Get full SID of user
# Output: admin S-1-5-21-1042354039-2475377354-766472396-1225
```

## Enum4linux

All-in-one SMB enumeration tool — automates multiple techniques.

```bash
# OS discovery
enum4linux -o 192.168.177.129

# User list
enum4linux -U 192.168.177.129
```

| Flag | Purpose |
|------|---------|
| `-o` | OS information |
| `-U` | User list |

## Key Takeaways

| Item         | Detail                                       |
| ------------ | -------------------------------------------- |
| SMB port     | 445 (modern), 139 (legacy NetBIOS)           |
| Null session | Anonymous access without credentials         |
| SID          | Security Identifier — unique user/group ID   |
| Rpcclient    | Powerful for user enumeration and SID lookup |
| Enum4linux   | One-tool solution for SMB recon              |
