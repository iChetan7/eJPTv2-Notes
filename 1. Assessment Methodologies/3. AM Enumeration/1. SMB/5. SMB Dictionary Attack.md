## Hydra Password Cracking

| Flag | Meaning |
|------|---------|
| `-p` | Single password (lowercase) |
| `-P` | Password list file (uppercase) |
| `-l` | Single login name (lowercase) |
| `-L` | Login name list file (uppercase) |

```bash
# Brute force with wordlist
hydra -l admin -P /usr/share/wordlists/rockyou.txt 192.168.177.129 smb

# Verify single credential
hydra -l msfadmin -p msfadmin 192.168.177.129 smb
```

## RID Enumeration

RID (Relative Identifier) = roll number for every user/group on the system.

### Rpcclient
```bash
rpcclient -U "" -N 192.168.177.129
rpcclient $> enumdomusers
rpcclient $> enumdomgroups
```

### Enum4linux
```bash
# Authenticated
enum4linux -r -u msfadmin -p msfadmin 192.168.177.129

# Anonymous / null session
enum4linux -r 192.168.177.129
```

| Flag | Purpose |
|------|---------|
| `-r` | RID enumeration |
