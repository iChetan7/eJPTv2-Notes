## FTP Anonymous Login

Many FTP servers allow **anonymous access** without credentials — a common misconfiguration.

---

## Nmap Anonymous Check

```bash
nmap 192.168.177.129 --script ftp-anon
```

Checks if anonymous login is enabled on the target FTP server.

---

## Manual Anonymous Login

```bash
ftp 192.168.177.129
```

| Prompt | Input |
|--------|-------|
| `Name` | `anonymous` |
| `Password` | `anonymous` or any email |

---

## Key Takeaways

| Item               | Detail                               |
| ------------------ | ------------------------------------ |
| Default FTP port   | 21                                   |
| Anonymous username | `anonymous`                          |
| Anonymous password | `anonymous` or blank/email           |
| `ftp-anon` script  | Automated check for anonymous access |
| Risk               | Unauthenticated file read/write      |
