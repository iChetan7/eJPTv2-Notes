## SSH — Secure Shell

Used for **remote administration**. Provides a secure shell to interact with remote machines over an **encrypted** channel. Runs on **TCP port 22**.

---

## Enumeration

```bash
# Version and OS detection
nmap -p22 192.168.177.129 -sV -O
```

---

## SSH Connection

```bash
# Standard connection
ssh root@192.168.177.129

# Force older algorithms (legacy systems)
ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedAlgorithms=+ssh-rsa root@192.168.177.129
```

## NSE SSH Scripts

### Enumerate Algorithms
```bash
nmap 192.168.177.129 -p22 --script ssh2-enum-algos
```

Lists all SSH algorithms supported by the server.

---

### Retrieve Host Key
```bash
nmap 192.168.177.129 -p22 --script ssh-hostkey --script-args ssh_hostkey=full
```

Displays full RSA/DSA/ECDSA host key fingerprints.

---

### Enumerate Auth Methods
```bash
nmap 192.168.177.129 -p22 --script ssh-auth-methods --script-args "ssh.user=msfadmin"
```

Shows authentication methods available for a specific user (password, publickey, keyboard-interactive, etc.).

---

## Flag Reference

| Flag/Option                            | Purpose                                 |
| -------------------------------------- | --------------------------------------- |
| `-o HostKeyAlgorithms=+ssh-rsa`        | Enable legacy RSA host key algorithm    |
| `-o PubkeyAcceptedAlgorithms=+ssh-rsa` | Accept legacy RSA public keys           |
| `ssh2-enum-algos`                      | List supported SSH algorithms           |
| `ssh-hostkey`                          | Extract server host key                 |
| `ssh-auth-methods`                     | Discover auth methods for user          |
| `ssh.user=`                            | Specify username for auth method script |
