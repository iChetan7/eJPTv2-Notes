## HTTP Enumeration

HTTP is for hosting websites. **IIS** (Internet Information Services) is Microsoft's web server software.

---

## Quick Recon

```bash
# Identify technologies
whatweb 192.168.177.129

# Check HTTP response
http 192.168.177.129

# Directory brute force (~4600 words default)
dirb http://192.168.177.129

# Terminal browser
browsh --startup-url http://192.168.177.129
```

---

## Nmap HTTP Scripts

```bash
# Enumerate directories and files
nmap 192.168.177.129 -p80 -sV -O --script http-enum

# Grab HTTP headers
nmap 192.168.177.129 -p80 -sV -O --script http-headers

# Check allowed methods on specific path
nmap 192.168.177.129 -p80 -sV -O --script http-methods --script-args http-methods.url-path=/webdav

# Scan for WebDAV misconfigurations
nmap 192.168.177.129 -p80 -sV -O --script http-webdav-scan --script-args http-methods.url-path=/webdav
```

---

## Script Reference

| Script | Purpose |
|--------|---------|
| `http-enum` | Directory and file enumeration |
| `http-headers` | Retrieve HTTP response headers |
| `http-methods` | Discover allowed HTTP methods |
| `http-webdav-scan` | Detect WebDAV vulnerabilities |
