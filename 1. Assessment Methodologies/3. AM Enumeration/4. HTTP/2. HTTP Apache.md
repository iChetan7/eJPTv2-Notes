## Tool Reference

| Tool/Module    | Purpose                                    |
| -------------- | ------------------------------------------ |
| `http_version` | Identify web server version                |
| `curl`         | Fetch and display page content             |
| `wget`         | Download files from server                 |
| `browsh`       | Modern terminal browser                    |
| `lynx`         | Classic text-based browser                 |
| `brute_dirs`   | Metasploit directory brute forcer          |
| `dirb`         | Directory brute force with custom wordlist |
| `robots_txt`   | Enumerate robots.txt entries               |
| `dir_scanner`  | Metasploit directory scanner               |
