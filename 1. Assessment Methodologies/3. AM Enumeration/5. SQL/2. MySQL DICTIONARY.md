## MySQL Enumeration

MySQL runs on **TCP port 3306**. Enumeration focuses on version detection, credential testing, data extraction, and privilege escalation vectors.

---

## Nmap MySQL Scripts

### Version Detection
```bash
nmap -p3306 192.168.177.129 -sV
```

### Check Blank Root Password
```bash
nmap -p3306 192.168.177.129 --script=mysql-empty-password
```

### Extract Server Info
```bash
nmap -p3306 192.168.177.129 --script=mysql-info
```

### List User Accounts
```bash
nmap -p3306 192.168.177.129 --script=mysql-users --script-args="mysqluser='root',mysqlpass=''"
```

### List Databases
```bash
nmap -p3306 192.168.177.129 --script=mysql-databases --script-args="mysqluser='root',mysqlpass=''"
```

### Extract Config Variables
```bash
nmap -p3306 192.168.177.129 --script=mysql-variables --script-args="mysqluser='root',mysqlpass=''"
```

### CIS Security Audit
```bash
nmap -p3306 192.168.177.129 --script=mysql-audit --script-args="mysql-audit.username='root',mysql-audit.password='',mysql-audit.filename=/usr/share/nmap/nselib/data/mysql-cis.audit"
```

### Dump Password Hashes
```bash
nmap -p3306 192.168.177.129 --script=mysql-dump-hashes --script-args="username='root',password=''"
```

### Execute Arbitrary SQL Query
```bash
nmap -p3306 192.168.177.129 --script=mysql-query --script-args="query='select * from mysql.user;',username='root',password=''"
```

```sql

SHOW VARIABLES LIKE 'secure_file_priv';

-- Test write to common directories
```

## Script Reference

| Script | Purpose | Auth Required |
|--------|---------|---------------|
| `mysql-empty-password` | Test blank root password | No |
| `mysql-info` | Server version and capabilities | No |
| `mysql-users` | List user accounts | Yes |
| `mysql-databases` | List databases | Yes |
| `mysql-variables` | Config variables | Yes |
| `mysql-audit` | CIS security audit | Yes |
| `mysql-dump-hashes` | Extract password hashes | Yes |
| `mysql-query` | Execute custom SQL | Yes |
