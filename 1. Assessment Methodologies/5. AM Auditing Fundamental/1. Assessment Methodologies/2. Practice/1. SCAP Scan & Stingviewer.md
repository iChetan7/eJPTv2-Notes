## SCAP Scan & Stingviewer

**SCAP** — Security Content Automation Protocol. A standardized approach for maintaining security of enterprise systems using specifications for expressing and manipulating security data.

---

## SCAP Components

| Component | Purpose |
|-----------|---------|
| **CCE** | Common Configuration Enumeration — system config identifiers |
| **CPE** | Common Platform Enumeration — product naming scheme |
| **CVE** | Common Vulnerabilities and Exposures — vulnerability dictionary |
| **CVSS** | Common Vulnerability Scoring System — severity ratings |
| **XCCDF** | Extensible Configuration Checklist Description Format — policy benchmarks |
| **OVAL** | Open Vulnerability and Assessment Language — vulnerability assessment |

---

| Tool | Type |

| SCC (Security Compliance Checker) | DISA tool |
| Stigviewer | STIG viewer and checklist tool |

## STIG — Security Technical Implementation Guides

| Item | Detail |
|------|--------|
| Publisher | DISA (Defense Information Systems Agency) |
| Purpose | Hardening guidelines for DoD systems |
| Format | XCCDF benchmarks |
| Viewer | Stigviewer for reading and tracking compliance |

---

## SCAP Scan Workflow

```bash
# OpenSCAP example
oscap xccdf eval --profile xccdf_org.ssgproject.content_profile_standard --results results.xml /usr/share/xml/scap/ssg/content/ssg-ubuntu2004-ds.xml

# View STIG checklist
# Open Stigviewer → Import STIG → Apply to target system → Track findings
```
