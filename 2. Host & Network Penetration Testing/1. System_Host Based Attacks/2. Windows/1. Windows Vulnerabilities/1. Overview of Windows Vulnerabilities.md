## Windows Vulnerability

A **weakness or mistake** in the Windows system that attackers can exploit to:

- Gain unauthorized access
- Steal files or passwords
- Install viruses or ransomware
- Take control of the device

---

## Frequently Exploited Windows Services

| Service | Port | Description |
|---------|------|-------------|
| Microsoft IIS | 80/443 | Web server for hosting websites |
| WebDAV | 80/443 | Web-based file authoring and versioning |
| SMB/CIFS | 445 | File and printer sharing |
| RDP | 3389 | Remote desktop access |
| WinRM | 5985/5986 | Windows Remote Management (HTTP/HTTPS) |

---

## Key Takeaways

| Item | Detail |
|------|--------|
| Common entry points | Web services, file sharing, remote access |
| SMB port 445 | EternalBlue, other major exploits target this |
| RDP port 3389 | Brute force and BlueKeep vulnerability |
| WinRM | PowerShell remoting, lateral movement vector |
| WebDAV | Often misconfigured, allows file upload/execution |
