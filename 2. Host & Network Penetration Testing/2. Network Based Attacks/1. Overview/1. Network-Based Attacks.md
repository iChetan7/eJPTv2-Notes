**What is a Network-Based Attack?**

- Attacks that deal with network traffic and network services
- OS-agnostic — targets the network itself, not the OS
- Includes: ARP, DHCP, SMB, FTP, Telnet, SSH attacks

**Types covered:**

- Brute force / password spraying on network services
- **Man-in-the-Middle (MITM)** — intercept traffic between two systems
- Packet sniffing / traffic analysis

**How MITM Works:**

- Old hub days — all traffic was broadcast, easy to sniff
- Modern switches — traffic goes only to intended recipient
- To intercept now: use **ARP Poisoning** or **Promiscuous Mode**

**ARP Poisoning:**

- Broadcast fake ARP packets claiming your MAC = target IP
- Inside network, routing is done by MAC address
- Traffic meant for another machine gets redirected to you
