**What is Wireshark?**

- GUI packet capture and analysis tool
- Captures and displays network traffic in real time
- Saves captures as `.pcap` or `.pcapng` files
- Command line version = **TShark**

---

## Commands

**Nmap aggressive scan (discover network)**

```bash
nmap -A 192.168.1.0/24
# Runs port scan + service scan + OS detection + default scripts
```

**Start Wireshark capture:**

- Select correct interface (eth1 for lab network)
- Click shark fin icon to start
- Stop when done

**Save capture:**

```
File → Save As → rando.pcapng
```

**Useful Wireshark Settings:**

```
View → Time Display Format → UTC Date and Time    # standard format
View → Name Resolution → Resolve Physical/Network addresses
View → Coloring Rules                              # color coded packets
```

**Add source/destination port columns:**

```
Edit → Preferences → Columns → Add new columns
→ Source Port + Destination Port
```

**Display Filters:**

```
not arp                          # hide ARP packets
not arp and not tcp              # show only ICMP etc.
ip.addr == 192.112.112.3         # filter by IP address
tcp.stream eq 3005               # filter specific TCP stream
http                             # show only HTTP traffic
```

**Follow a stream:**

```
Right-click packet → Follow → TCP Stream / HTTP Stream
# Shows full conversation in readable format
```

**Export HTTP objects (images, files):**

```
File → Export Objects → HTTP
# Saves all files transferred over HTTP
```

**Statistics:**

```
Statistics → Protocol Hierarchy    # what protocols are in capture
Statistics → Conversations         # which machines talked to each other
Statistics → Endpoints             # all endpoints seen
```

---

**⚠️ Key Points**

- Always capture on **correct interface** (eth1 not eth0)
- HTTP traffic = readable in Wireshark (plain text)
- HTTPS traffic = encrypted = looks like gibberish
- ARP works at Layer 2 — no TCP conversation, just MAC + IP
- TShark = command line Wireshark — better for remote/scripted use
- `.pcapng` = next-gen packet capture format 🔥
