**Key WiFi Concepts:**

- 8q02.11 = WiFi protocol
- Beacon frame = AP broadcasting its presence (SSID info here)
- BSSID = MAC address of access point
- SSID = network name

**Security Types to identify:**

| Tag              | Security     |
| ---------------- | ------------ |
| No WPA/RSN tag   | Open network |
| WPA tag          | WPA-PSK      |
| RSN tag (tag 48) | WPA2-PSK     |
| WPS tag          | WPS enabled  |

---

**Wireshark Filters for WiFi**

```wireshark
# Find all beacon frames
wlan.fc.type_subtype == 0x0008

# Find open networks (beacon + no RSN tag)
wlan.fc.type_subtype == 0x0008 && !wlan.tag.number == 48

# Find specific SSID
wlan contains Home_Network

# Find specific BSSID
wlan.bssid == e8:de:27:16:87:18

# Packets by MAC address (sent or received)
wlan.addr == e8:de:27:16:87:18

# Data frames only
wlan.fc.type_subtype == 0x0020

# Association response
wlan.fc.type_subtype == 0x0001
```

---

**What to look for in Beacon frame:**

```
Beacon frame
  └── Tagged parameters
        ├── SSID          → network name
        ├── DS Parameters → current channel
        ├── WPA tag       → WPA security
        ├── RSN (tag 48)  → WPA2 security
        └── WPS tag       → WPS enabled
```

**Key Notes**

- Open network = no WPA/RSN tags in beacon
- `contains` filter = case sensitive in Wireshark
- Channel info = DS Parameters inside beacon
- WPS enabled = easy attack target
