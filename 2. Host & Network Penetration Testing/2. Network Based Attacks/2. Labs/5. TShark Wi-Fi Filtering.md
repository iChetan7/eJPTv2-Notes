**Key Commands from the Lab**

**Filter all Wi-Fi traffic:**

```bash
tshark -r wifi.pcap -Y "wlan"
```

**View de-authentication packets:**

```bash
tshark -r wifi.pcap -Y "wlan.fc.type_subtype == 0x000c"
```

**Filter WPA handshake packets:**

```bash
tshark -r wifi.pcap -Y "eapol"
```

**Print SSID and BSSID of all beacon frames:**

```bash
tshark -r wifi.pcap -Y "wlan.fc.type_subtype == 0x0008" -T fields -e wlan.ssid -e wlan.bssid
```

**Find BSSID of a specific SSID:**

```bash
tshark -r wifi.pcap -Y "wlan.ssid == LazyArtist" -T fields -e wlan.bssid
```

**Find devices that received de-auth messages:**

```bash
tshark -r wifi.pcap -Y "wlan.fc.type_subtype == 0x000c" -T fields -e wlan.ra
```

**Identify a device by MAC via HTTP User-Agent:**

```bash
tshark -r wifi.pcap -Y "wlan.ta == 5c:51:88:31:a0:3b && http" -T fields -e http.user_agent
```

→ Result: **Motorola Moto G3, Android 6.0**
