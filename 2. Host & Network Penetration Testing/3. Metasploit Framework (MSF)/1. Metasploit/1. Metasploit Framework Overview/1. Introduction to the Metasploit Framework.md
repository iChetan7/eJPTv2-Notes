## What is Metasploit?

An open-source penetration testing and exploitation framework —> the de facto standard for pentesters and security researchers. It automates stages of the pentest lifecycle: reconnaissance, exploitation, and post-exploitation, and hosts one of the largest databases of publicly tested exploits.

## Brief History

|Year|Event|
|---|---|
|2003|Created by HD Moore, written in Perl|
|2007|Rewritten in Ruby|
|2009|Acquired by Rapid7|
|2019|Metasploit 5.0 released|
|2020|Metasploit 6.0 released|

## Key Terminology

**Interface** —> how you interact with MSF (console, CLI, GUI)

**Module** —> a piece of code that performs a specific task

**Vulnerability** —> a weakness/flaw in a system that can be exploited

**Exploit** —> code that takes advantage of a vulnerability

**Payload** —> code delivered to the target by an exploit; provides a shell or executes commands

**Listener** —> waits for an incoming reverse shell connection from the target

## Interfaces Available

| Interface             | Description                                           |
| --------------------- | ----------------------------------------------------- |
| **MSFconsole**        | Primary all-in-one CLI; most commonly used            |
| **MSF CLI**           | Deprecated in 2015; used for automation/scripting     |
| **Community Edition** | Web-based GUI for network discovery                   |
| **Armitage**          | Free Java-based GUI; visual wrapper around MSFconsole |

The course focuses on **MSFconsole** as the primary interface which is standard in real-world pentesting.
