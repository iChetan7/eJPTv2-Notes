**Basic Scans**

```bash
# default scan
nmap <target-ip>

# if host seems down (Windows blocks ping)
nmap -Pn <target-ip>

# service version detection
nmap -Pn -sV <target-ip>

# OS detection + service versions
nmap -Pn -sV -O <target-ip>

# all together + save to XML (for Metasploit import)
nmap -Pn -sV -O <target-ip> -oX scan_results.xml
```

**Import into Metasploit**

```bash
msfconsole
db_import /path/to/scan_results.xml
hosts        # see discovered hosts
services     # see discovered services
```

**Common Ports to note:**

|Port|Service|
|---|---|
|80|HTTP|
|135|MSRPC|
|139|NetBIOS|
|445|SMB|
|3389|RDP|

**Key Notes**

- `-Pn` = skip ping probe (needed for Windows targets)
- `-sV` = service version detection
- `-O` = OS detection
- `-oX` = output as XML (required for Metasploit import)
- XML import lets Metasploit auto-suggest exploit modules
