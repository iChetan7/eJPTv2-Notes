## What is a Web Server?

Software that serves website data via HTTP. Common examples: Apache, Nginx, Microsoft IIS. Uses **port 80** (HTTP) or **port 443** (HTTPS).

## Web Server Enumeration Workflow

**Step 1 — Identify web server version**

```bash
use auxiliary/scanner/http/http_version
run
# Result: Apache 2.4.18 on Ubuntu Linux
```

**Step 2 — Enumerate HTTP headers**

```bash
use auxiliary/scanner/http/http_header
# HTTP_METHOD: HEAD or GET
# IGN_HEADER: list headers to exclude
run
# Result: content-type, last-modified, server banner
```

**Step 3 — Analyze robots.txt**

```bash
use auxiliary/scanner/http/robots_txt
run
# Result: Disallowed: /data, /secure
```

The `robots.txt` file tells search engines what not to index — developers sometimes accidentally reveal sensitive directories here.

**Step 4 — Directory brute force**

```bash
use auxiliary/scanner/http/dir_scanner
# DICTIONARY: auto-set (can customise)
# PATH: / (root) or specific subdirectory
run
# Result: /secure (401 auth required), /data, /downloads, /webdav, /webmail
```

**Step 5 — File brute force**

```bash
use auxiliary/scanner/http/files_dir
# EXT: leave blank or specify e.g. php
run
# Result: index.html, tests.php, code.c
```

**Step 6 — Enumerate Apache users**

```bash
use auxiliary/scanner/http/apache_userdir_enum
set USER_FILE /usr/share/metasploit-framework/data/wordlists/common_users.txt
run
echo "rooty" > user.txt
# Result: user "rooty" found  
```

**Step 7 — Brute force protected directory**

```bash
use auxiliary/scanner/http/http_login
set AUTH_URI /secure
set USER_FILE /root/user.txt      # file with "rooty"
set PASS_FILE /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt
set VERBOSE false                  # only show successes
run
```

## Key Modules Summary

|Module|Purpose|
|---|---|
|`http_version`|Web server version + OS detection|
|`http_header`|HTTP response headers|
|`robots_txt`|Hidden/sensitive directories|
|`dir_scanner`|Directory brute force|
|`files_dir`|File brute force|
|`apache_userdir_enum`|Valid username enumeration|
|`http_login`|Brute force HTTP authentication|
